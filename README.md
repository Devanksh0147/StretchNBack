# StretchNBack

A lightweight Windows utility to quickly switch resolutions, create presets, and revert safely.

## Features

- One-click resolution switching
- Preset system for different use cases (gaming, normal, etc.)
- Safe revert system to prevent display issues
- Built-in theme support (Light / Dark / System)
- QRes bundled (no external setup required)
- Standalone EXE (no Python required)

---

## Download

👉 Download the latest version from Releases

## Usage

1. Run the application (Admin required)
2. Select a resolution and refresh rate
3. Click Apply
4. Confirm within 15 seconds or it will revert automatically

---

## Presets

- Save your favorite configurations
- Quickly switch between them anytime

---

## Notes

- Works on Windows (64-bit only)
- Antivirus may flag EXE (false positive due to PyInstaller)
- Run as Administrator for full functionality
